## Getting Started

### Local Development

1. Clone this repository
2. Open `index.html` in your web browser
3. That's it! No build process required.

### Hosting on GitHub Pages

1. Create a new repository on GitHub
2. Push your code to the repository:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
   git push -u origin main
   ```
3. Go to repository Settings > Pages
4. Select "main" branch as source
5. Your site will be live at `https://YOUR_USERNAME.github.io/YOUR_REPO/`

### Hosting on Namecheap

1. Upload all files to your Namecheap hosting via FTP or File Manager
2. Ensure `index.html` is in the root directory (or public_html)
3. Your site will be accessible at your domain

## Customization

### Colors

Edit the CSS variables in `css/style.css`:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
    --dark: #1e1b4b;
    --light: #f8fafc;
}
```

### Content

Edit the text content directly in `index.html`. The structure is organized into sections:
- Navigation
- Hero Section
- About Section
- Services Section
- Contact Section
- Footer

### Animations

All animations are defined in `css/animations.css`. You can:
- Adjust timing and duration
- Create new animations
- Modify existing effects
